What is Petit UI ?

Petit UI is a lightweight UI library made with Petit.css for building small projects without using a larger framework. There is a Github repository is available here.

Installation

You can download Petit UI here. You can also add it to your projects with Npm or Yarn : 


  # npm
  npm install petit-ui

  # yarn
  yarn add petit-ui

Button

Petit UI comes with basic buttons types : primary, secondary, info, success, danger, warning. Buttons styles can be applied to <a>, <button> and <input type="submit"> elements.
